# Diccionarios RNDC 2026-07-09

Fuente: exportaciones descargadas por el usuario desde el portal oficial RNDC el 9 de julio de 2026.

## Archivos

- `official-data-dictionary-rndc.xls`: exportacion original del diccionario de datos. Aunque usa extension `.xls`, el contenido es texto tabulado ISO-8859-1.
- `official-error-dictionary-rndc.xls`: exportacion original del diccionario de errores. Aunque usa extension `.xls`, el contenido es texto tabulado ISO-8859-1.
- `data-dictionary-rndc.tsv`: copia UTF-8 tabulada para lectura automatizada.
- `error-dictionary-rndc.tsv`: copia UTF-8 tabulada para lectura automatizada.
- `SHA256SUMS`: huellas de integridad de los cuatro archivos.

Los originales no deben editarse. Una actualizacion del portal debe guardarse en una carpeta nueva con su propia fecha y huellas.
